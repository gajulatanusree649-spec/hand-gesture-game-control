# Hand Gesture Controlled Game System, a real-time touchless interaction solution built using computer vision and AI concepts.

This project leverages OpenCV for video processing and MediaPipe for accurate hand tracking. Using a webcam, the system captures live video input and detects 21 hand landmarks to analyze finger positions and movements in real time.

Based on the number of fingers detected, the system recognizes specific gestures and maps them to keyboard inputs using a key simulation module. For example, a closed fist is interpreted as a braking action, while an open hand triggers acceleration. Other gestures act as neutral states, ensuring stable and controlled interaction. The system is optimized to run with low latency, providing a smooth and responsive user experience.

This project demonstrates how traditional input methods can be replaced with intuitive, touchless controls, enhancing user interaction and accessibility. It highlights the integration of Artificial Intelligence, Computer Vision, and Human-Computer Interaction to build smart and interactive systems.

# Key Highlights:

Real-time hand detection and tracking
Finger counting using landmark-based analysis
Gesture-to-keyboard mapping
Lightweight and efficient implementation
Works with a standard webcam setup

# Tech Stack: Python, OpenCV, MediaPipe

# Applications: Gaming, virtual simulations, touchless interfaces, assistive technologies, and future AR/VR systems.
